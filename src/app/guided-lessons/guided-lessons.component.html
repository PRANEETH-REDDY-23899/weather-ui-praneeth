Guided Lessons

Machine Learning is an increasingly relevant topic in science, policy and society at large. Understanding the basics of it will be useful.

For this lesson, read this introduction on machine learning. Then consider the questions below. You may want to find other sources (one example) for more details about the topics below.

Discussion Questions

Question 1: How can machine learning be used ethically?
Question 2: What are the advantages and disadvantages of using machine learning?
Question 3: What types of problems do you think machine learning would help to solve?

Now that you have a basic understanding of machine learning, you may be interested in using our web application. However, we recommend that you familiarize with our data set before doing so. It's important to understand the data so you better understand how to interpret the machine learning results.

For this lesson, read this read through the understanding the data page.

Discussion Questions

Question 1: What are the limitations of this data set? How could the dataset be improved?
Question 2: How does variance impact our analysis of a model's results.

Feature Engineering is an important topic relating to improving model training and performance.

For this lesson, read through this introduction on feature engineering (you can skip the sections on specific feature engineering tools). Then read our feature engineering page to learn about our process of feature engineering.

Discussion Questions

Question 1: What are the tradeoffs that we consider when feature engineering?
Question 2: How does feature engineering improve model training?
Question 3: How do you evaluate if your feature engineering approach is successful or not?

There are many different machine learning models, each with their own advantages. For our web application, we have used the following Auto Regression, ARIMA, SARIMA, LSTM, RNN, Multiple Linear Regression, Support Vector Regression and Random Forest.

For this lesson, read the Machine Learning Model Info page. Don't worry about understanding the exact method of each one. If you were to use a model, then you would want to have that deeper understanding. Instead, just focus on the differences between them and how they can vary.

Discussion Questions

Question 1:Choose one model to learn more about. What are the advantages of this model compared to others?
Question 2:Which model seems the least useful to you? Why? If you aren't sure, choose a model and research its potential drawbacks.

Now that you have an understanding of machine learning, you should try out our application!

For this lesson, go back to the home page, and play around with the application. Consider the discussion questions as you do so.

Discussion Questions

Question 1: How does the window size impact the model performance?
Question 2: Do you feel that one model is outperforming others?
Question 3: Do you think that the provided visualization is useful for understanding the performance of models? What would you change?
